## Install

```
pipenv install
```

## Run
```
pipenv run python3 notes --list
pipenv run python3 notes --read-today
pipenv run python3 notes --read-tomorrow
pipenv run python3 notes --edit-today
pipenv run python3 notes --edit-tomorrow
pipenv run python3 notes --copy-today
pipenv run python3 notes --copy-tomorrow
```
